WHAT IS MACHINE LEARNING
Machine learning is a vital subset of artificial intelligence that focuses on developing algorithms capable
of learning from data and making decisions or predictions without being explicitly programmed. It
leverages large datasets to recognize patterns, enabling systems to adapt and improve over time. There
are three main types of machine learning: supervised, unsupervised, and reinforcement learning.
 In supervised learning, models are trained on labeled datasets, where the correct output is known,

 making it ideal for tasks like classification and regression. Unsupervised learning, on the other hand,
 deals with unlabeled data, where the algorithm identifies hidden patterns or groupings within the data,
commonly used in clustering and dimensionality reduction. Reinforcement learning involves an agent
 that learns to make decisions by interacting with an environment, receiving feedback through rewards or

 penalties, which is essential for tasks like game playing or autonomous driving.
 Machine learning has vast applications across industries, from personalized recommendations in e-
 commerce and fraud detection in finance to disease prediction in healthcare and autonomous vehicles.
 Its ability to process and analyze complex data has revolutionized many sectors, driving innovation and
 enabling more intelligent, data-driven decision-making processes.

WHAT IS WEB DEVELOPMENT
Web development refers to the process of creating, building, and maintaining websites and web
applications. It involves various aspects such as web design, web content development, client-
side/server-side scripting, and network security configuration. Web development can be divided into
three main categories: front-end development, back-end development, and full-stack development..
 Front-end development focuses on the part of the website that users interact with, using languages like

HTML, CSS, and JavaScript to create responsive and visually appealing interfaces. Back-end development
involves the server-side of the website, managing databases, server logic, and application programming
interfaces (APIs) using languages like Python, PHP, Ruby, or Node.js.
 Full-stack developers work on both the front-end and back-end, handling the entire development

process from start to finish. Web development also includes testing, performance optimization, and
ensuring the site is accessible and user-friendly across different devices and browsers.
